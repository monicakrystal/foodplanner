# Purpose 

a cute website for me to plan my meals & save my favorite recipes

# Installation

1. Clone the repo  
   `git clone <repo-url>`

2. Install dependencies  
   `npm install`

3. Run the development server  
   `npm run dev`

4. Open in browser  
   http://localhost:3000


# Tech Stack 

- Next.js 15 (App Router)
- React 19
- TypeScript
- CSS Modules
